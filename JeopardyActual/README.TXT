PROJECT TITLE: Jeopardy
PURPOSE OF PROJECT: To create a fun and interactive game similar to that of the popular game show, Jeopardy
VERSION or DATE:
HOW TO START THIS PROJECT:
AUTHORS: Morgan Kay and Carolina Phillips
USER INSTRUCTIONS: Type in a number to indicate the row you wish to choose  a  category from, followed by the column. A question will then appear, to which you can type your answer. Your score will be adjusted accordingly. It is then the other player's turn. Continue until all questions have been answered and a winner determined.
